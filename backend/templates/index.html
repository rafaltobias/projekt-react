Visit Tracker API

This is the API documentation for the Visit Tracker application. Use the React frontend for a better user experience.

Visit Endpoints

POST /track

Track a new visit

Request Body:

{
    "url": "https://example.com",
    "tag_id": 1  // optional
}

Stats Endpoints

GET /api/stats

Get visit statistics

GET /api/exportStats

Export visit statistics as CSV

Tag Endpoints

POST /api/tags

Create a new tag

Request Body:

{
    "name": "Tag Name",
    "description": "Tag Description"  // optional
}

GET /api/tags

Get all tags

GET /api/tags/:id

Get tag by ID

DELETE /api/tags/:id

Delete tag by ID
